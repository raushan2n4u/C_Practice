
Program 1Assignment
1. Given an array of size n, find the majority element. The majority element is the element that appears more than n/2 times. If no majority element exists then return the minimum value of the integer.

Example 1:
Input: [3,2,3]
Output: 3

Example 2:
Input: [2,2,1,1,1,2,2]
Output: 2

Example 3:
Input: [2,3,1,1,1,2,2]
Output: -2147483648

Write an efficient C code for this and analyze its time and space complexity.
